# Neuro Mirror AI

Real-time facial expression analysis and emotion recognition system using neural networks for applications in UX research, mental health, and HCI.

## Features

- Real-time face detection and tracking
- Multi-class emotion classification (happy, sad, angry, surprised, neutral, fear, disgust)
- Expression intensity scoring
- Session recording and analysis export
- Webcam and video file input

## Tech Stack

TypeScript, TensorFlow.js, React, Canvas API

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/neuro-mirror-ai.git
cd neuro-mirror-ai
npm install
```

### Usage

```bash
npm run dev
```
